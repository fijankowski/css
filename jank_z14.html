Sposoby definiowania (określanie) kolorów

Sposób zapisu Składnia teoretyczna Przykład zastosowania
Heksadecymalny (szesnastkowy) color: #RRGGBB; DIV { color: #FFFFFF; }
Definicja RGB (liczbowo) color: rgb(R, G, B); (zakres 0-255) DIV { color: rgb(255, 100, 13); }
Nazwa własna koloru color: nazwa_koloru; DIV { color: red; }
Definicja RGB (procentowo) color: rgb(R%, G%, B%); (zakres 0%-100%) DIV { color: rgb(10%, 40%, 78%); }
