1. Składnia teoretyczna

<head>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>

2. Gdzie wstawiany jest styl zewnętrzny?

Styl zewnętrzny wstawiany jest w sekcji head dokumentu HTML za pomocą znacznika link. Może występować wiele razy.

3. Najprostsza forma

<link rel="stylesheet" href="style.css">

4. Ile arkuszy można dołączyć?

Można dołączyć wiele arkuszy CSS. Należy zachować kolejność, ponieważ ostatni ma najwyższy priorytet.

5. Konflikty

W przypadku konfliktu stylów pierwszeństwo ma styl zdefiniowany później.

6. Edytor tekstowy

Pliki CSS można tworzyć i edytować w zwykłym edytorze tekstu, np. Notatnik.

7. Edytor CSS

Można używać specjalnych edytorów CSS, które ułatwiają pisanie kodu.

8. Rozszerzenie pliku

Pliki arkuszy stylów mają rozszerzenie .css

9. Zalety

1. Możliwość zmiany wyglądu wielu stron jednocześnie
2. Oddzielenie struktury od wyglądu

10. Uwagi

CSS nie toleruje błędów, nie pokazuje błędów oraz może działać różnie w różnych przeglądarkach.
