Użycie identyfikatorów ID - teoria

Pojęcie Znaczenie pojęcia (Uzupełnił Filip Jankowski)
Co znaczy skrót ID? Skrót pochodzi od słowa "Identifier" (Identyfikator).
Jaki jest cel stosowania ID? Służy do jednoznacznego zidentyfikowania oraz ostylowania jednego, konkretnego i unikalnego elementu na stronie WWW.
Jakie jest znaczenie znaku # Znak hash (#) w arkuszu stylów CSS informuje przeglądarkę, że następująca po nim nazwa jest selektorem identyfikatora (ID).
Jak następuje wywołanie ID? Wywołanie następuje w kodzie HTML poprzez dopisanie do tagu atrybutu id="nazwa_identyfikatora".
Uwagi na temat stosowania ID 1. Identyfikator musi być unikalny – dane ID może wystąpić tylko RAZ w całym dokumencie HTML.
2. Jeśli dany styl ma się powtarzać dla wielu elementów, należy użyć atrybutu 'class' zamiast 'id'.
3. Nazwa ID nie może zaczynać się od cyfry i nie powinna zawierać polskich znaków ani spacji.
